## Set Up

1. Create your environment

    `python3 -m venv .venv`

2. Activate it

    `source .venv/bin/activate`

3. Install the requirements

    `pip install -r requirements.txt`

4. On VSCode:
   1. Do `ctrl+shift+p`
   2. Click on `Python: Select Interpreter`
   3. Select the new created `.venv`
